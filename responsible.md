# responsible

**rdfs:label:** responsible  
**rdf:type:** owl:ObjectProperty (recursive)

## Definition
Establishes hierarchies of Economic Agents with higher-level agents (role: **superior**) accountable for the activities of lower-level agents (role: **subordinate**).

## Domain / Range
- **Domain:** Economic Agent  
- **Range:** Economic Agent

## Notes
Used to reflect organizational hierarchies; widely used in enterprise software coding schemes.
