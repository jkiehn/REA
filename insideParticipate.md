# insideParticipate

**rdfs:label:** insideParticipate  
**rdf:type:** owl:ObjectProperty

## Definition
Participation association linking an **inside** Economic Agent (of the object enterprise) to an occurrence of an Economic Event.

## Domain / Range
- **Domain:** Economic Event  
- **Range:** Economic Agent

## Notes
Represents the internal party in the trading-partner view.
