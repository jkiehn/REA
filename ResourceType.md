# ResourceType

**rdfs:label:** ResourceType  
**rdf:type:** owl:Class

## Definition
A classification that characterizes kinds of Economic Resources (e.g., 'Finished Good', 'Raw Material', 'Service Capacity') for reuse across commitments and events.

## Intended Meaning
Provides the conceptual template for resources that are increased/decreased by events and reserved by commitments.
