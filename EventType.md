# EventType

**rdfs:label:** EventType  
**rdf:type:** owl:Class

## Definition
A classification that characterizes kinds of Economic Events (e.g., 'Sale', 'Receipt', 'Production Run'), providing reusable semantics for commitments and event instances.

## Intended Meaning
Encapsulates event semantics (roles, typical attributes, policies) that can be referenced by `Commitment` and `Economic Event` instances.
