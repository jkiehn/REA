# outsideParticipate

**rdfs:label:** outsideParticipate  
**rdf:type:** owl:ObjectProperty

## Definition
Participation association linking an **outside** Economic Agent (external to the object enterprise) to an occurrence of an Economic Event.

## Domain / Range
- **Domain:** Economic Event  
- **Range:** Economic Agent

## Notes
Represents the external party in the trading-partner view.
